Instalación de WildAI

Antes de ejecutar WildAI, asegúrate de tener instalado Python 3 y los siguientes paquetes:

  • pandas
  • scikit-learn (sklearn)
  • joblib
  • seaborn
  • numpy
  • matplotlib (matplotlib.pyplot)
  • imbalanced-learn (imblearn)

Se pueden instalar fácilmente con el siguiente comando en tu terminal:

pip install pandas scikit-learn joblib seaborn numpy matplotlib imbalanced-learn
